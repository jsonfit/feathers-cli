## usage

```
feat --help
Usage: feat <command> [options]

Options:
  -V, --version                  output the version number
  -h, --help                     display help for command

Commands:
  create|c [options] <app-name>  create a new project powered by feathers-cli
  help [command]                 display help for command

Examples
  feat create hello-world

Run feat <command> --help for detailed usage of given command.
```
